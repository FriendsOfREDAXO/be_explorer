REDAXO-AddOn: Backend Explorer
================================================================================

Dateiexplorer für REDAXO.
Dateien direkt auf dem Server über die Weboberfläche erstellen, bearbeiten, kopieren, etc. Farbliche Quellcode-Hervorhebung im Editiermodus.
